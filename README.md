# SQNative

> **IAB:** v0.1.0 (Foundation Layer)  
> **Note:** This project is currently in early development. This release contains the core Driver Layer.

**SQNative** is a toolkit for SQLite in Node.js.

It is built exclusively on top of the native `node:sqlite` module (introduced in Node.js v22.5), rejecting the legacy ecosystem of C++ bindings, `node-gyp` compilation, and Python dependencies.
